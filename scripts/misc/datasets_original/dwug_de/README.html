Description

This data collection contains diachronic Word Usage Graphs (WUGs) for German. Find a description of the data format, code to process the data and further datasets on the WUGsite.

We provide additional data under misc/:

  • dwug_de_sense: a subset of DWUG DE was annotated with classical word sense definitions (DWUG DE Sense, see misc/dwug_de_sense/data/*/judgments_senses.csv). This folder provides clusterings, change scores and inferred binary semantic proximity labels for this subset. The clusters, statistics and proximity labels under maj_2 and maj_3 were derived from the sense annotation by removing instances where not at least 2/3 annotators agree on the label. Note that the binary proximity labels under misc/dwug_de_sense/data/*/maj_[2/3]/judgments.csv ('0' for different sense, '1' for same sense) were derived from the sense annotation, and not directly judged by humans (in contrast to other WUG data sets). Note that consequently also the scores EARLIER, LATER and COMPARE are not calculated directly from human judgments (as for other WUG data sets), but from the inferred binary proximity labels. Please find the code deriving the clusters, change scores and proximity labels in the WUG repository. The data set is described in more detail in Schlechtweg (2022).

See previous versions for additional testsets.

Please find more information on the provided data in the paper referenced below.

Version: 2.3.0, 15.12.2022. Contains additional clusterings, change scores and binary semantic proximity labels derived from DWUG DE Sense. Important: Version 2.0.0 extends previous versions with one more annotation round and new clusterings.

Beschreibung

Diese Datensammlung enthält diachrone Wortverwendungsgraphen (WUGs) fürs Deutsche. Eine Beschreibung des Datenformats, Code zur Verarbeitung der Daten und weitere Datensätze finden Sie auf der WUGseite.

Wir stellen zusätzliche Daten unter misc/ zur Verfügung:

  • dwug_de_sense: Eine Teilmenge von DWUG DE wurde mit klassischen Wortbedeutungsdefinitionen annotiert (DWUG DE Sense, siehe misc/dwug_de_sense/data/*/judgments_senses.csv). Dieser Ordner enthält Clusterings, Bedeutungswandelwerte und abgeleitete binäre 'semantic proximity labels' für diese Teilmenge. Die Cluster, Statistiken und Proximity-Labels unter maj_2 und maj_3 wurden aus der klassischen Bedeutungsannotation abgeleitet, indem Instanzen entfernt wurden, bei denen nicht mindestens 2/3 Annotatoren übereinstimmen. Beachten Sie, dass die binären Proximity-Labels unter misc/dwug_de_sense/data/*/maj_[2/3]/judgments.csv ('0' für verschiedene Bedeutungen, '1' für dieselbe Bedeutung) aus klassischen Bedeutungsannotation abgeleitet und damit nicht direkt von Menschen annotiert wurden (im Gegensatz zu anderen WUG-Datensätzen). Beachten Sie weiter, dass die Werte EARLIER, LATER und COMPARE somit nicht direkt aus menschlichen Annotationen berechnet wurden (wie bei anderen WUG-Datensätzen), sondern aus den abgeleiteten binären Labels. Den Code, der die Cluster, Wandelwerte und Proximity Labels berechnet, finden Sie im WUG-Repository. Der Datensatz wird in Schlechtweg (2022) näher beschrieben.

Zusätzliche Testdaten finden Sie in früheren Versionen.

Weitere Informationen zu den bereitgestellten Daten entnehmen Sie bitte dem unten angegebenen Papier.

Version: 2.3.0, 15.12.2022. Enthält zusätzliche Clusterings, Bedeutungswandelwerte und Proximity-Labels für DWUG DE Sense. Wichtig: Version 2.0.0 erweitert frühere Versionen um eine weitere Annotationsrunde und neue Clusterings.
